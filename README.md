# 🗺️ VWorld PRO: 토지이용규제 및 공시지가 시각화 솔루션

VWorld API를 활용하여 대한민국 전 국토의 지적정보, 공시지가, 그리고 토지이용규제 정보를 실시간으로 조회하고 시각화하는 웹 애플리케이션입니다. 교육용 및 실무용으로 모두 활용 가능하도록 설계되었습니다.

## 🚀 주요 기능

### 1. 지능형 필지 속성 (Parcel Insight)
- **실시간 공시지가 조회**: 마우스 호버 시 해당 필지의 최신 공시지가를 즉시 표시합니다.
- **자동 단위 변환**: 공시지가를 `1㎡당 가격`과 `평당 가격`으로 자동 환산하여 보여줍니다. (천원 단위)
- **토지이용규제 통합 조회**: 용도지역, 용도지구, 용도구역(가축사육제한 등) 정보를 여러 레이어에서 동시에 추출하여 결합 표시합니다.

### 2. 프리미엄 UI/UX
- **리사이징 사이드바**: 왼쪽 메뉴 바의 너비를 자유롭게 조절하거나, 완전히 숨겨서(접기/펴기) 광활한 지도 화면을 확보할 수 있습니다.
- **레이어 카탈로그**: 브이월드의 수많은 정보를 검색하고, `+` / `-` 버튼으로 지도에 즉시 추가하거나 제거할 수 있습니다.
- **Pinned Area (즐겨찾기 ★)**: 자주 사용하는 레이어를 별도로 고정하여 관리할 수 있습니다.

### 3. 하이브리드 배포 (Local & Cloud)
- **로컬 수업용**: `vworld_server.py`를 통해 오프라인/로컬 환경에서 안정적인 프록시 통신을 지원합니다.
- **클라우드 배포용**: Vercel Serverless Function(`api/proxy.js`)을 탑재하여 별도의 서버 설정 없이 즉시 배포 가능합니다.

## 🛠️ 설치 및 실행 방법

### 로컬 환경 (수업용)
1. **서버 실행**: Python이 설치된 환경에서 프록시 서버를 가동합니다.
   ```bash
   python vworld_server.py
   ```
2. **웹 접속**: 브라우저에서 `index.html`을 엽니다. (또는 Live Server 사용)

### 클라우드 배포 (Vercel)
1. GitHub 저장소에 코드를 푸시합니다.
2. [Vercel](https://vercel.com)에 로그인 후 해당 저장소를 임포트합니다.
3. 배포된 주소(`.vercel.app`)를 [브이월드 개발자 센터](https://www.vworld.kr)에 도메인 등록합니다.

## 🔑 설정 방법
- `index.html` 상단의 `API_KEY` 변수에 본인의 브이월드 API 키를 입력하세요.
- VWC(브이월드) 정책에 따라 사용하시는 도메인(localhost 또는 배포 주소)이 키와 일치해야 합니다.
